## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## API

## Auth

```
# Register new user [POST]
http://localhost:9595/app/auth/register

{
  "firstname": "string",
  "lastname": "string",
  "username": "string",
  "password": "string"
}

# Login user [POST]
http://localhost:9595/app/auth/login

{
  "username": "admin@admin.com",
  "password": "admin"
}

# Get user data [GET]
Header: Bearer [token]
http://localhost:9595/app/profile

```
